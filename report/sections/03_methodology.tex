\section{Methodology}

\subsection{Research Design}

本研究采用小样本混合方法设计，将在线问卷的描述性结果与半结构化访谈材料结合起来，考察亚洲女大学生如何理解校园健身房空间、身体意象压力与阻力训练回避之间的关系。
原设计为顺序解释型混合方法：先用问卷建立整体趋势，再用质性访谈解释这些趋势。
实际数据收集完成后，量化部分包括问卷平台导出的题项汇总表，以及后续补齐的逐名文本版答卷矩阵；质性部分为 3 名受访者的一对一半结构化访谈实录。
因此，本研究在描述性统计之外，补充进行小样本探索性的个体层面信度、相关和分组比较，但不进行多变量回归、中介模型或因果推断。

这一调整并不改变研究问题本身，而是改变证据的强度和表达方式。
量化结果用于说明哪些空间感受、回避原因和干预偏好在样本中更突出，并初步检验空间压迫、媒介审美内化与自由重量区回避之间的双变量关系；质性材料则用于解释这些数字背后的情境机制。
本研究仍以 SEAM 关于公共运动评价焦虑和回避的测量思路 \parencite{levinson2013seam}、SATAQ-4 关于社会文化审美内化的测量思路 \parencite{schaefer2015sataq4}、以及健身房性别化空间研究 \parencite{turnock2021gyms,rendall2024gym} 为理论参照，但不声称使用了完整原版量表。

\subsection{Participants and Data Sources}

问卷共有 34 人开始作答。
第 1 题显示 33 人为在校大学生；第 2 题显示其中 32 人自我认同为女性。
第 3 题之后的核心分析样本因此为 $N = 32$ 名女性在校大学生。
年龄分布集中在 18--20 岁（87.50\%），另有 21--23 岁 3 人（9.38\%）和 24 岁及以上 1 人（3.13\%）。
由于问卷并未单独测量民族、国籍或族裔身份，本文中的``亚洲女大学生''主要指本研究所处中文语境和亚洲校园情境中的女性在校学生；相关结论不应被外推为所有亚洲女性学生的普遍经验。

质性材料来自 3 名受访者的半结构化访谈，访谈主题与问卷一致，围绕校园健身房经验、自由重量区感受、被观看/被评价体验、社交媒体审美内容、运动目标和对学校支持措施的建议展开。
3 名受访者的经验存在差异：一名较常使用健身房和力量训练区，一名本学期因课程开始尝试力量训练，一名主要在宿舍训练、仅短暂进入过健身房。
这种差异有助于比较``相对熟悉者''、``新近进入者''和``边缘/回避者''对同一空间的不同理解。

\subsection{Survey Instrument}

问卷共 29 题。
第 1--2 题为纳入筛选题；第 3--9 题收集年龄、运动频率、校园健身房使用、常用区域、力量训练经验和运动目标。
第 10--15 题为 1--5 分滑动条，测量自由重量区的空间压迫与训练焦虑，包括布局不易接近、男性较多时不自在、镜面与公开可见带来的评价感、动作不标准担忧、男性主导感、以及拥挤/器械集中造成的不想停留。
第 16--18 题测量主动回避自由重量区的频率、回避原因和应对方式。
第 19--23 题测量社交媒体使用和审美内容影响，其中第 21--23 题为 1--5 分滑动条。
第 24 题测量训练目标自我效能；第 25--27 题测量三类潜在干预的接受度，包括减少刻板印象、增加半私密分区或视觉遮挡、以及提供女性初学者力量训练 workshop。

与 proposal 中原计划的 60--100 份逐名问卷和 5--6 人质性小组访谈相比，最终可用样本规模更小。
不过，后续补齐的文本版 Excel 文件提供了每名参与者在第 3 题之后的逐题作答，因此本研究同时报告每题有效样本数、选项频数、滑动条题项均值、分段比例和个体层面构念均值。
滑动条由问卷平台导出为五个分段：1--1.8、1.9--2.6、2.7--3.4、3.5--4.2、4.3--5。
本文将 3.5--5 两个分段合并称为``高分段''，用于描述某一判断被较强认可的比例。
个体层面构念分数按相关题项的平均值计算：第 10--15 题为空间压迫/训练焦虑，第 21--23 题为社交媒体审美内化，第 24 题为训练自我效能单题，第 25--27 题为干预偏好指数。
由于第 25--27 题分别指向不同干预方式，本文将其视为偏好题组，而不是严格的反映式心理量表。

\subsection{Qualitative Procedure and Analysis}

访谈采用半结构化提纲，但允许受访者按个人经历展开。
问题从具体场景进入，例如第一次进入健身房的感觉、有氧区与力量区的差别、是否担心动作被评价、社交媒体是否影响运动选择、以及希望学校提供怎样的课程或空间支持。
访谈材料整理为 QA 格式后匿名化，分析时只以 P1、P2、P3 指代受访者。

质性分析采用主题分析思路。
第一步反复阅读 3 份实录，标记与研究问题相关的意义单元；第二步围绕空间可见性、新手门槛、男性主导氛围、策略性协商、审美影响和干预需求进行初始编码；第三步将相近编码合并为主题，并用代表性引语检验主题是否能解释量化发现。
由于样本只有 3 人，主题分析不追求饱和度或广泛推广，而是用于解释问卷中较高的空间压迫、回避和干预偏好。

\subsection{Data Analysis}

量化数据使用 Python 进行整理和汇总。
分析脚本首先解析问卷平台导出的汇总 Excel 表格，将题目、选项频数、滑动条均值和分段频数拆分为结构化表格；随后读取逐名文本版答卷矩阵，生成去身份化的参与者得分表。
在个体层面，本文计算空间压迫/训练焦虑和社交媒体审美内化的 Cronbach's $\alpha$，用 Spearman's $\rho$ 检验构念得分与自由重量区回避频率之间的探索性相关，并用 Mann--Whitney $U$ 检验比较自由重量区常用者与非常用者、有无系统力量训练经验者之间的差异。
所有 $p$ 值均作为小样本探索性证据理解，未用于强因果判断。
质性数据同样通过 Python 解析 Markdown 访谈文本，生成 QA 表、主题编码表和代表性引语表。

量化与质性整合发生在结果和讨论部分。
本文先报告问卷中的高频现象，再用访谈解释这些现象如何被受访者体验为镜面可见性、男性占多数、新手动作羞耻、器械知识门槛和策略性进入。
这种整合方式适合当前数据形态：双变量统计可以提示变量之间是否同向变化，但不能证明统计因果关系；访谈材料则帮助形成关于校园健身房性别化进入门槛的情境化证据链。

\subsection{Ethical Considerations}

本研究讨论身体意象、被观看感和健身房不适，可能触及羞耻、焦虑或自我评价压力。
因此，问卷和访谈均应建立在自愿参与、匿名处理和可随时退出的原则上。
访谈材料在报告中只使用去身份化引语，并避免呈现能够识别个人身份的细节。
由于本项目为课程研究，结果只用于课程论文写作，不作个体诊断或学校管理评价。
